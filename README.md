# Exploring-Accidental-Car-Sales-with-ML-models-and-Image-Classification-with-TensorFlow-and-Keras

🚗📸 Exploring Second-Hand Car Sales & Image Classification with Deep Learning
📌 Overview

This project combines machine learning on structured data and deep learning on image data to explore two domains:
Analysis of second-hand car sales data using ML techniques
Image classification using CNN models built with TensorFlow and Keras
The project demonstrates end-to-end workflows including preprocessing, modeling, evaluation, and optimization.

🎯 Objectives

Analyze second-hand car sales data to uncover trends and patterns
Build predictive models using machine learning techniques
Develop a robust image classification model using CNNs
Improve model performance through data augmentation and tuning

🚗 Car Sales Analysis (Machine Learning)

Explored dataset containing second-hand car attributes
Performed data cleaning and preprocessing
Applied ML techniques to identify patterns in pricing and features
Generated insights to support data-driven decisions

📸 Image Classification (Deep Learning)

Built Convolutional Neural Networks (CNNs) using TensorFlow & Keras
Trained models on a flower image dataset 

Applied:
Data Augmentation
Dropout (to reduce overfitting)
Hyperparameter tuning
Evaluated model performance and improved accuracy across experiments

🧠 Models & Techniques Used

Machine Learning (Car Data):
Regression / Classification techniques
Data preprocessing & feature engineering

🌸Deep Learning (Images):

CNN (Convolutional Neural Networks)
TensorFlow & Keras
Regularization (Dropout)
Data Augmentation
